A WordPress plugin that allows users to upload CSVs of addresses, and then query them using a custom endpoint of the WordPress REST API.

== Installation ==
Download [the latest release](https://github.com/ardouglass/address-mapper-wp/releases/latest) (the `address-mapper.zip file`) and install the archive through your plugins page.
